👶🧠 BabyVLM Leaderboard 2026

Results on the DevCV Toolbox — the 11-benchmark evaluation suite from BabyVLM-V2: Toward Developmentally Grounded Pretraining and Benchmarking of Vision Foundation Models. Scores are accuracy (0–100); the Overall Average is the mean of the 11 primary metrics (missing benchmarks count as 0). Evaluation is run by the organizers on the held-out (IRB-restricted) SAYCam test sets.

No submissions have been published yet.

🍼 Submit a model

Because the test data is held out, you don't run the test set yourself. A submission is two things: a model.py lmms-eval wrapper + a checkpoint. Fill in this form and click Submit — it opens a pre-filled submission on our GitHub, where your entry is recorded. The organizers then run the held-out DevCV test set on an L40S GPU and, once approved, your row appears above.

⚠️ = required · 👶 = required for the challenge · 🔹 = optional. Submitting opens a pre-filled GitHub issue for you to review and post. See the submission guide for the PR alternative and the reference model.py.
